# Hermes Obsidian MVP

A minimal Obsidian plugin that opens a sidebar chat and talks to Hermes through ACP by spawning `hermes acp` locally.

## Goals

- Sidebar chat inside Obsidian
- Local ACP transport to Hermes
- Stream assistant text into the UI
- Use the vault path as the working directory

## Status

Early MVP scaffold.
